El siguiente proyecto, fue realizado en un 95% con html y css, por lo tanto, los slideshow, carousel y animaciones 
son completamente estilos, utilizando como apoyo, el JS para darle funcionalidad a los tabs unicamente.

La página web, emula una ficha técnica. Se utilizó el logo de las páginas amarillas para darle mas identidad al sitio.

El tiempo de desarrollo del sitio tomó entre unas 16 a 20 horas.
